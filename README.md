

## 👋 Introduction
This API provides the backend services for a **Finance Dashboard System**, enabling users to manage and access financial records securely and efficiently. Developed using **Spring Boot**, it supports robust role-based access control (RBAC), allowing different user types—such as analysts, viewers, and admins—to interact with financial data according to their specific permissions.

---

## 🚀 Key Capabilities

* **Financial Records Management:** Complete CRUD (Create, Read, Update, Delete) functionality for financial entries.
* **User and Role Management:** Maintain user profiles, assign roles, and control granular permissions.
* **Summary & Analytics Endpoints:** Retrieve aggregated financial insights designed for high-level dashboard visualizations.
* **Secure Access:** Industry-standard authentication and authorization mechanisms to ensure data protection.

---

## 🛠 Tech Stack
* **Backend Framework:** Spring Boot (Java)
* **Data Format:** JSON
* **Documentation:** Postman / REST Client
